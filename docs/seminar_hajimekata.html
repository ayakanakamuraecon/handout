研究の始め方

テーマの探し方、良い問いの条件、RQ（リサーチ・クエスチョン）の立て方について説明します。

01テーマの探し方

研究テーマの出発点は、自分の身の回りのことへの「引っかかり」や疑問です。「答えが分かるわけない」「無理」と最初から決めつけず、まず実際に調べたうえで、自分の問いを絞り込んでいきます。

まずは漠然とでもよいので、研究の範囲を決めることが必要です（例：教育政策・人口動態など）。

大切なのは「知りたいこと」と「使えそうなデータ」が両方あることです。以下は経済学でよく扱われるテーマの例です。必ずしもこの中から選ぶ必要はありません。

労働・雇用

  • 女性就業率が高い地域にはどのような特徴があるか
  • 非正規雇用比率が高い地域では、平均所得は低いか

教育

  • 大学進学率が高い地域ほど、平均所得は高いか
  • 教育支出の大きい地域ほど、進学率は高いか

人口・地域

  • 人口減少が大きい地域ほど、失業率は高いか
  • 地方と都市部で、若者の生活環境にどのような違いがあるか

家計・生活

  • 家計所得が高い地域ほど、消費支出は大きいか
  • 物価が高い地域ほど、生活の余裕は小さいか

福祉・医療

  • 保育所整備率が高い地域ほど、女性就業率は高いか
  • 医師数が多い地域ほど、平均寿命は長いか

地域経済

  • 観光客数が多い地域ほど、地域所得は高いか
  • 製造業比率の高い地域ほど、賃金は高いか
テーマを考えるときのポイント
  • 「おもしろそう」で終わらず、XとYを数字で置けるか考える
  • まずは都道府県別など、比較しやすい単位から考える
  • 広すぎるテーマは避ける
  • 公開データで無理なく扱える範囲に絞る

02良い問いの条件

研究テーマを考えるときは、「おもしろそう」だけで終わらせず、データで確かめられる問いになっているかを意識します。良い問いを立てることは、良い研究のための第一歩であり、核心でもあります。

〇 良い問いの条件
  • 何を知りたいかが1文で言える
  • XとYがある程度はっきりしている
  • 数字で置きかえられる
  • 比較する単位がある
  • 使えそうなデータがある
✕ 悪い問いの特徴
  • 広すぎる
  • 抽象的すぎる
  • 数字で測れない
  • 何と何を比べるのか分からない
  • 原因を強く言いすぎている

悪い問いを、良い問いに直す

悪い問いの例問題点改善例
少子化はなぜ起こるのか 範囲が広すぎる。原因が多すぎる 保育所整備率が高い地域ほど、女性就業率は高いか
住みやすい地域とは何か 住みやすさが抽象的。何で測るか分からない 家賃が高い地域ほど、平均所得は高いか
教育は大事か 価値判断になっている。分析の形になっていない 大学進学率が高い地域ほど、平均所得は高いか
観光地は地域を活性化させるか 「活性化」があいまい。「させる」という表現が強い 観光客数が多い地域ほど、宿泊業や飲食業の割合は高いか

03RQの立て方

問いは疑問文にします。論文の基本は、自分の立てた問いに、裏付けとなる資料・データを示して答えることです。問いはWhat（〜とは何か）ではなく、Why（なぜ〜なのか）、How（どうすれば解決できるか）、Should（〜をどうするべきか）で考えます。What型の問いは説明で終わってしまうため、解決策を考える問いのほうが望ましいとされています。

まず目指したい問いの形

X が大きい地域ほど、Y はどうか
X と Y にはどのような関係があるか
X が異なると、Y に違いは見られるか
問いを立てる際に留意すること
  • どのような研究方法を使いたいか（事例研究か統計分析か）。研究方法は問いによって決まることが多いため、問いを立てる段階で見通しを立てておく
  • データ収集や観察を行うことができそうか
  • 分析や観察を行う単位をどうするか

問いを考えるときのチェックポイント

Xは何か
Yは何か
XとYは数字で置けるか
何を1つの単位として比べるか
そのデータは公開されているか
