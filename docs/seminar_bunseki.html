分析と解釈

回帰分析の基本的な考え方と、結果の読み方についてまとめています。

01回帰分析は何のためにやるのか

回帰分析は、X が Y にどの程度関連しているかを数値で示すための手法です。ただし、関連の強さを示すものであって、それ単独で因果関係を証明するものではありません。
  • Y（被説明変数）：説明したい結果。RQ で書いた「結果として見たいもの」
  • X（説明変数）：Y に影響を与えていると考える要因。RQ で書いた「要因として見たいもの」

RQ 検討シートで書いた X と Y が、そのまま回帰式の変数になります。

02回帰式の形

最も単純な回帰式は次の形です。

$$Y = \beta_0 + \beta_1 X + u$$
  • β0（切片）：X が0のときの Y の値
  • β1（傾き）：X が1単位増えたときに Y がどれだけ変化するかを表す

R では次のように書いて係数を推定します。

lm(Y ~ X, data = df)

コントロール変数を入れる場合

X 以外にも、Y と相関していそうな要素（コントロール変数）がある場合は、式に追加できます。コントロール変数を入れることで、その影響（例：所得や年齢構成など）を取り除いたうえで、X と Y の関係を見ることができます。ただし、変数を増やしすぎるとサンプルサイズに対して変数が多くなりすぎます。特に単年の都道府県データ（n=47）のような小標本では、多くても5個程度に抑えるのが目安です。

ダミー変数の扱い方

カテゴリ（地域、年次、性別など）を回帰式に入れたい場合は、ダミー変数（0か1の値を取る変数）に変換して使います。
例：「2026年かどうか」を表すダミー変数は、2026年なら1、それ以外なら0を取る。
R では factor() で変数をカテゴリ型に変換すれば、lm() が自動的にダミー変数を作って分析します。

lm(Y ~ X + factor(year), data = df)

都道府県×年のパネルデータでは、地域ダミー・年ダミーを入れることで、地域ごとの固有の特徴や、年ごとの全国的な変化の影響を考慮できます。

03結果の読み方

結果表が出てきても、数字を読むだけでは不十分で、それぞれの数字が何を意味しているかを考えることが大切です。次の順番で読みます。

1
係数の符号→大きさを見る
2
p値を確認する
3
R2を確認する
4
結果から言えること・言えないことを考える

① 係数

係数は、X が1単位変化したときに、Y がどのくらい変化するかを表します。例えば、係数が0.5のとき、X が1増えると、Y は平均して0.5増える関係があります。

ダミー変数の係数は、その変数が1のグループは、0のグループと比べて Y が平均してどれだけ違うかを表します。例えば、「2026年ダミー」の係数が0.5のとき、2026年は基準の年と比べて Y が平均して0.5大きい、という意味になります。

  • プラスかマイナスか
  • 大きさはどのくらいか
  • 単位は何か
注意：係数がプラスでもマイナスでも、それだけで因果とは言えません。

② p値

p値は、「X と Y には本当は関係がないのに、たまたまデータ上で関係があるように見えているだけ」という可能性がどのくらいあるかを示す目安です。この値が小さいほど、見られた関係が偶然である可能性が低いと考えます。一般に、p値が0.05未満であれば、統計的に有意（偶然とは考えにくい）と判断します（より厳しく0.01、緩く0.10を基準にすることもあります）。

  • 統計的に有意か
  • ただし、有意かどうかだけで判断しない
注意：
  • 有意でなくても、「何も分からない」わけではありません。──関係が見られなかったこと自体が一つの結果であり、「X と Y には明確な関係がなさそうだ」という情報になります。
  • 有意であっても、「重要である」とは限りません。──データの数が多いと、ごく小さな関係でも「有意」と出やすくなります。有意かどうかと、関係の大きさ（係数の大きさ）は別のことなので、両方を見る必要があります。

③ R2

R2は、モデルがデータにどのくらい当てはまっているか（フィットしているか）を表す指標です。0から1の値をとり、1に近いほど当てはまりがよいことを示します（Y のばらつきのうち、モデルで説明できる割合を表します）。

  • 値が高いほど、モデルの当てはまりはよい
  • ただし、高ければ良い研究というわけではない
注意：R2が高くても、因果関係が示されたことにはなりません。

04発表で使いたい表現・避けたい表現

回帰分析は、結果を正しく解釈し、適切な表現で報告することが大切です。同じ結果でも、言い方によって「言いすぎ」になってしまうことがあります。

〇 使いたい表現
  • X が高い地域ほど、Y も高い傾向が見られた
  • X と Y には正の関係が見られた
  • 回帰分析の結果、X の係数はプラスであった
  • 統計的に有意な関係が見られた
✕ 避けたい表現
  • X が Y を増やした
  • X が Y の原因である
  • この結果で証明された

※どのような表現を使う場合でも、回帰分析だけでは因果関係までは言えないことに注意してください。

05回帰結果を見るときの注意

数字だけで終わらず、単位を意識する
現実に照らして不自然ではないか考える
有意かどうかだけに注目しない
相関と因果を混同しない
最低限できるようになってほしいこと
  • 係数の符号と大きさを説明できる
  • p値と R2 の意味を大まかに説明できる
  • 結果から言えることと言えないことを区別できる
